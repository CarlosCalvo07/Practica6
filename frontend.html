¿Que es FRONTEND?

Es la parte visual de una página web con la que los usuarios interactúan.

¿Como funciona FRONTEND?

Funciona mediante HTML, CSS y JavaScript para estructurar, diseñar e interaccionar en la web.

¿Caracteristicas de FRONTEND?

Es dinámico, interactivo y se adapta a diferentes dispositivos.
